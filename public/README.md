# HTML Frontend for Archeology Game

This directory contains the browser-based frontend for the Archeology in Mesopotamia game.

## Files

- `index.html` - Main game interface
- `style.css` - Game styling
- `gameController.js` - Game logic and UI controller

## How to Play

1. Open `index.html` in a web browser
2. Hire personnel (workers, archaeologists, linguists) using the left sidebar
3. Select an excavation method (Surface Collection, Excavation, or Trench)
4. Click on tiles in the site view to select them for excavation
5. Click "Start Excavation" to begin
6. Wait for the excavation to complete
7. Identify and sell artifacts to earn money
8. Discover new sites and continue exploring!

## Features

- Interactive tile-based excavation
- Artifact discovery and identification
- Resource management
- Multiple excavation methods
- Layer-based site exploration
- Inventory system

## Browser Compatibility

Works in modern browsers that support ES6 JavaScript (Chrome, Firefox, Safari, Edge).
